Connect4 AI - Java

Connect-4 with AI and Graphical User Interface

This is a Connect-4 AI written in Java and implementing a Minimax Algorithm with Alpha-Beta Pruning. With difficulty = 4 and under 100,000 distinct games, this AI beats a random AI 99.983% of the time and loses 0.013% of the time.

Alien Invasion

This is a Sidescroller written in C++ using the Qt Creator and an Event-Driven Programming Scheme. This Project was completed in collaboration with Darren Atallah, Weiyu Huang, and Peter Liu. Among other things, I was responsible for handling user input, player abilities, and optimizing program efficiency.
